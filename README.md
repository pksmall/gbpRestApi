# gbpRestApi
Internship REST API by GeekBrains 

app's using python3.5+

## run app 

#### git clone
```$ git clone https://github.com/pksmall/gbpRestApi.git```


#### install requirements

```$ pip install -r requirements.txt```

#### app start
```$ python app.py```

#### get data
```$ curl 127.0.0.1:5000/v1/persons/rank ```
